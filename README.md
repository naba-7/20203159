# 오픈소스SW개론 과제2

top, ps, jobs, kill 명령어 조사

------------------------------------------------------------------------

# top
  - 프로세스 실시간 모니터링 top은 Linux 시스템의 CPU, 메모리, 프로세스 사용량을 실시간으로 모니터링하는 명령어

  __기본 사용__
  ```
  top
  ```

  주요 단축키
  - q : 종료
  - P : CPU 사용량 기준 정렬
  - M : 메모리 기준 정렬
  - T : 실행 시간 기준 정렬
  - k : 프로세스 종료
  - 1 : CPU 코어별 사용량 표시

------------------------------------------------------------------------

# ps
  - 프로세스 정보 출력 ps는 현재 실행 중인 프로세스를 출력하는 명령어

  __기본 사용__
  ```
  jobs
  ```

  자주 쓰는 옵션
  - ps -e : 모든 프로세스 출력
  - ps -f : 전체 포맷 출력
  - ps aux : 전체 정보 표시
  - ps -ef : 계층 구조 포함 전체 프로세스 표시
 
------------------------------------------------------------------------

# jobs
  - 백그라운드 작업 확인 jobs는 현재 쉘에서 실행 중이거나 정지된 작업을 보여주는 명령어
  - &, Ctrl + Z 와 함께 사용됨

  __기본 사용__
  ```
  jobs
  ```

  ex)
  __백그라운드 실행__
  ```
  sleep 100 &
  ```
  __작업 목록 확인__
  ```
  jobs
  ```
  __포그라운드로 가져오기__
  ```
  fg %1
  ```

  출력 구조
  * [번호] : 작업 번호
  * Running / Stopped : 상태
  * Command : 실행 중인 명령어

  ------------------------------------------------------------------------

  # kill 
    - 프로세스 종료 / 시그널 전달 kill은 프로세스에 시그널을 전달하여 종료시키는 명령어이다.
