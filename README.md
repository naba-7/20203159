# 1. top
  - top은 Linux 시스템의 CPU, 메모리, 프로세스 사용량을 실시간으로 모니터링하는 명령어

  __기본 사용__
  ```
  top
  ```

  주요 단축키
  - q : 종료
  - P : CPU 사용량 기준 정렬
  - M : 메모리 기준 정렬
  - T : 실행 시간 기준 정렬
  - k : 프로세스 종료
  - 1 : CPU 코어별 사용량 표시

------------------------------------------------------------------------

# 2. ps
  - ps는 현재 실행 중인 프로세스를 출력하는 명령어
  - top이 실시간 모니터링이면, ps는 한 번만 출력되는 스냅샷 정보

  __기본 사용__
  ```
  jobs
  ```

  자주 쓰는 옵션
  - ps -e : 모든 프로세스 출력
  - ps -f : 전체 포맷 출력
  - ps aux : 전체 정보 표시
  - ps -ef : 계층 구조 포함 전체 프로세스 표시
 
------------------------------------------------------------------------

# 3. jobs
  - jobs는 현재 쉘에서 실행 중이거나 정지된 작업을 보여주는 명령어
  - &, Ctrl + Z 와 함께 사용됨

  __기본 사용__
  ```
  jobs
  ```

  ex)
  __백그라운드 실행__
  ```
  sleep 100 &
  ```
  __작업 목록 확인__
  ```
  jobs
  ```
  __포그라운드로 가져오기__
  ```
  fg %1
  ```

  출력 구조
  * [번호] : 작업 번호
  * Running / Stopped : 상태
  * Command : 실행 중인 명령어

------------------------------------------------------------------------

# 4. kill 
  - kill은 프로세스에 시그널을 전달하여 종료시키는 명령어
  - 주요 용도는 프로세스 종료

  __기본 사용__
  ```
  kill PID
  ```
  __정상 종료 요청__
  ```
  kill -15 PID
  ```
  __강제 종료 요청__
  ```
  kill -9 PID
  ```

  * 자주 사용하는 Signal 목록
    - 1 : SIGHUP (프로세스 재시작)
    - 2 : SIGINT (Ctrl+C 인터럽트)
    - 9 : SIGKILL (강제 종료)
    - 15 : SIGTERM (정상 종료 요청)
    - 18 : SIGCONT (일시정지 작업 계속)
    - 19 : SIGSTOP (프로세스 정지)
